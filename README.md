**Technologies used:** 

- React
- React-dom
- Node.js
- react-bootstrap
- rc-progress
- React Reveal
- Newsapi
- Mongo
- Express
- Mongoose
- Heroku

**Installation:** 

1. Download the repo
2. Install dependencies `npm install`
3. Activate the app using `npm run start`
